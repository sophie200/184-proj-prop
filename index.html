Summary

The goal of our project is to create an all-in-one solution that helps users create 3D meshes using common technology available to most people. Our project makes use of the iPhone LiDAR to generate a point cloud, and convert the point cloud to a mesh using the Ball-Pivoting algorithm. The final output will be a 3D mesh file that the user can open up in any 3D software and postprocess for printing, etc.
